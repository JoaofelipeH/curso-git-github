# 🎓 Curso de Git e GitHub

Repositório destinado ao armazenamento de notas, exercícios e projetos práticos desenvolvidos durante o curso de **Git e GitHub**. O objetivo principal é dominar o controle de versão e a colaboração em projetos de software.

---

## 🚀 Tecnologias Utilizadas

* **Git**: Sistema de controle de versão distribuído.
* **GitHub**: Plataforma de hospedagem de código e colaboração.
* **Markdown**: Linguagem de marcação para documentação.

## 📂 Conteúdo do Repositório

Neste repositório, estão documentados os seguintes conceitos:
* **Comandos Básicos**: `git init`, `git add`, `git commit`, `git status` e `git log`.
* **Ramificações (Branches)**: Gerenciamento, navegação e fusão (`merge`) de branches.
* **Resolução de Conflitos**: Práticas para lidar com divergências de código.
* **Trabalho Remoto**: Configuração de repositórios remotos, `push`, `pull` e `fetch`.
* **Boas Práticas**: Mensagens de commit claras e organização de fluxo de trabalho.
